# intellij-plugin-linpostfix

Android postfix plugin for AndroidStudio.

# Available templates

|   Postfix Expression  | Description                                                                                                            | Example                                |
| --------------------- | ---------------------------------------------------------------------------------------------------------------------- | -------------------------------------- |
| .new        | new a Class.                                                                             | new Test()
